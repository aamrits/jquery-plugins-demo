Owl Carousel

Description:

Owl Carousel 2 is a touch enabled jQuery plugin that lets you create a beautiful responsive carousel slider. It is fully customisable, supported in modern browsers and is responsive. Touch and drag support is also there. Below is the demo of this beautiful plugin.

Demo:

Set up:

This is the basic setup.

Step 1: Add the CSS files inside the <head> tag. Add JS files in the footer.


                                            <!-- owl Carousel CSS -->
                                            <link href="css/owl.carousel.min.css" rel="stylesheet" />
                                            <link href="css/owl.theme.default.min.css" rel="stylesheet" />
                                            
                                            <!-- Slim jquery -->
                                            <script src="js/jquery-3.2.1.slim.min.js"></script>
                                            <!-- Owl Carousel.js -->
                                            <script src="js/owl.carousel.min.js"></script>

NOTE: I am using Bootstrap v4.0 which uses slim jquery. This website is made on Bootstrap Framework. If you want to build your website in Bootstrap 4.0, then visit their website.

Step 2: Add the HTML.


                                            <div class="basic-carousel owl-carousel owl-theme">
                                                <div class="item">
                                                    <img src="img/img-1.jpg" class="img-fluid" alt="Image">
                                                </div>
                                                <div class="item">
                                                    <img src="img/img-2.jpg" class="img-fluid" alt="Image">
                                                </div>
                                                <div class="item">
                                                    <img src="img/img-3.jpg" class="img-fluid" alt="Image">
                                                </div>
                                                <div class="item">
                                                    <img src="img/img-4.jpg" class="img-fluid" alt="Image">
                                                </div>
                                                <div class="item">
                                                    <img src="img/img-5.jpg" class="img-fluid" alt="Image">
                                                </div>
                                                <div class="item">
                                                    <img src="img/img-6.jpg" class="img-fluid" alt="Image">
                                                </div>
                                                <div class="item">
                                                    <img src="img/img-7.jpg" class="img-fluid" alt="Image">
                                                </div>
                                                <div class="item">
                                                    <img src="img/img-8.jpg" class="img-fluid" alt="Image">
                                                </div>
                                            </div>

Step 3: Add the jQuery.


                                            (function($) {
                                                $(document).ready(function() {
                                                    //owl Carousel
                                                    $('.basic-carousel').owlCarousel({
                                                        loop:true,
                                                        margin:10,
                                                        nav:true,
                                                        responsive:{
                                                            0:{
                                                                items:1
                                                            },
                                                            600:{
                                                                items:1
                                                            },
                                                            1000:{
                                                                items:3
                                                            }
                                                        }
                                                    });
                                                });
                                            })(jQuery);

Owl Carousel (Vertical)

Description:

For implementing a vertical carousel, we need animate.css to be integrated with Owl Carousel 2. First, let us see the demo about how it looks.

Demo:

Set up:

This is the basic setup.

Step 1: Add the CSS files inside the <head> tag. You also need to add animate.css in the <head>. Add JS files in the footer.


                                            <!-- owl Carousel CSS -->
                                            <link href="css/owl.carousel.min.css" rel="stylesheet" />
                                            <link href="css/owl.theme.default.min.css" rel="stylesheet" />
                                            <!-- Animate CSS -->
                                            <link href="css/animate.css" rel="stylesheet" />
                                            
                                            <!-- Slim jquery -->
                                            <script src="js/jquery-3.2.1.slim.min.js"></script>
                                            <!-- Owl Carousel.js -->
                                            <script src="js/owl.carousel.min.js"></script>

Step 2: Add the HTML.

Add a new class .vertical-carousel in the div.


                                            <div class="vertical-carousel owl-carousel owl-theme">
                                                <div class="item">
                                                    <img src="img/img-1.jpg" class="img-fluid" alt="Image">
                                                </div>
                                                <div class="item">
                                                    <img src="img/img-2.jpg" class="img-fluid" alt="Image">
                                                </div>
                                                <div class="item">
                                                    <img src="img/img-3.jpg" class="img-fluid" alt="Image">
                                                </div>
                                                <div class="item">
                                                    <img src="img/img-4.jpg" class="img-fluid" alt="Image">
                                                </div>
                                                <div class="item">
                                                    <img src="img/img-5.jpg" class="img-fluid" alt="Image">
                                                </div>                                                
                                            </div>

Step 3: Add the jQuery.


                                            (function($) {
                                                $(document).ready(function() {
                                                    //owl Carousel
                                                    $(".vertical-carousel").owlCarousel({
                                                        loop: true,
                                                        autoplay: true,
                                                        items: 1,   //put "1" item for carousel to work
                                                        nav: true,
                                                        autoplayHoverPause: true,
                                                        animateOut: 'slideOutUp',
                                                        animateIn: 'slideInUp'
                                                    });
                                                });
                                            })(jQuery);

NOTE: You have to put items: 1. Then it will slide vertically one image at a time.

Reference:

  1. Download animate.css
